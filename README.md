# Ontario_Tech_ChatBot

## Description: 

### This is a chatbot built using IBM's Watson Assistant API as a proof of concept to show what can be deployed if needed.

## Requirements:
- ### Dependencies: 
  - body-parser
  - dotenv
  - express
  - normalize.css
  - watson-developer-cloud
- ### Dev Dependencies: 
  - @babel/core
  - @babel/preset-env
  - babel-loader
  - css-loader
  - file-loader
  - html-webpack-plugin
  - node-sass
  - style-loader
  - url-loader
  - webpack
  - webpack-cli
  - webpack-dev-server
  
## How to run:
  Run this command in a NodeJS enviroment `node start`.
